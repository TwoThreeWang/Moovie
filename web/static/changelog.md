## [v1.1.0] - 2025-11-04

全面优化搜索、缓存与播放体验，提升站点性能与响应速度

### ✨ 新增 (Added)

* 实现搜索缓存功能，显著提升搜索性能
* 添加每日定时任务清理低频与过期缓存
* 实现热门搜索词与最近搜索词展示
* 为搜索结果添加缓存控制头
* 添加轻量级 DTO 转换以减少 API 响应大小，提升接口响应速度
* 视频测速升级为综合视频质量测试
* 首页新增最近搜索区块并优化关键词展示样式
* 添加自动跳转下一集功能

### 🐞 修复 (Fixed)

* 修复爬虫数据解析时的类型转换问题
* 修复静态文件缓存头缺失导致的加载异常

### 🛠️ 变更/优化 (Changed/Improved)

* 重构爬虫服务，使用统一的数据结构
* 升级缓存服务并优化逻辑
* 优化搜索与播放页面的关键词上下文展示
* 新增多个资源采集站点
* 优化搜索结果的缓存与更新机制
* 优化热门页面分类展示与视频质量检测逻辑
* 改进视频质量检测功能并添加评分显示
* 优化数据库查询，仅选择必要字段，提升接口响应速度

### 🗑️ 移除 (Removed)

* 移除不再使用的 `Movies` 模型
* 移除旧的静态文件路由

### 💅 样式 (Style)

* 更新 CSS 样式，新增质量评分徽章样式
